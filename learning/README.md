# Learning

This folder contains the original learning materials from when this repo was a "daily practice" experiment.

## What Changed

I started with a plan: 15 minutes a day, small exercises, build Python and ML skills incrementally. Module 1 (Python foundations) worked fine. Then I hit a wall.

Small exercises didn't stick. I'd complete one, feel a small sense of accomplishment, and forget it by the next week. The investment was small, so the return was small. It felt like busywork.

What actually worked was building real projects:

- `ledger-db` taught me double-entry accounting by forcing me to implement it
- `payer-sync` taught me reconciliation, state machines, and idempotency by making me design a real system
- `will-they-pay` taught me ML pipelines by shipping a hackathon project

These projects took weeks, not minutes. But the learning stuck because I had to solve real problems, not toy exercises.

## What's Here

- `exercises/` - Python exercises from Module 1 (completed)
- `ROADMAP.md` - The revised learning plan (payments-focused)
- `SKILLS_TRACKER.md` - Project log
- `CONTRIBUTING.md` - How to add new content

## The Lesson

Small investment, small return. Large investment, large return.

If something feels pointless, it probably is. Find the version that feels meaningful.
