# Daily Upskill Repo (15 min every morning)

**Goal**: Consistent daily practice in Python, ML, Data Engineering & System Design.  
**Rule**: Open repo → 15 min max → commit & push every single day (no excuses).

**Structure**:
- `skills/` → reusable SKILL.md files (AI reads these automatically)
- `exercises/` → tiny daily implementations (one file per day)
- `ROADMAP.md` → what to learn next
- `SKILLS_TRACKER.md` → daily log

**Daily workflow (Cline/Aider)**:
1. Prompt: "Pick next task from ROADMAP.md, use relevant SKILL.md, put code in exercises/YYYY-MM-DD_*.py, update SKILLS_TRACKER.md"
2. Review changes
3. `git commit -m "feat: ..." && git push`
